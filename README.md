# kube-credential

## git clone https://github.com/Jeyssiva/kube-credential.git
## Issuance Service
## cd kube-credential/kube-issuance
## Install
    npm install
## Prisma generate + push
    npx prisma generate
    npx prisma db push

    db push will create the SQLite file and push the schema.
## Run locally (dev)
    npm run dev
